Location of the outputs JSON file, containing outputs and capabilities of another deployment. The file must be in the following format:

{
    "outputs": {
    	"<output_name>": "<output_value>",
    	"<output_name>": "<output_value>",
    	...
    },
    "capabilities": {
    	"<capability_name>": "<capability_value>",
    	"<capability_name>": "<capability_value>",
    	...
    }
}
