A JSON containing mapping between outputs/capabilities and inputs. The format must be of the following format: 

{
    "outputs": {
    	"<output_name>": "<input_name>",
    	"<output_name>": "<input_name>",
    	...
    },
    "capabilities": {
    	"<capability_name>": "<input_name>",
    	"<capability_name>": "<input_name>",
    	...
    }
}
